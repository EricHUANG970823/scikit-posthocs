Outliers API reference¶

outliers_iqr(x[, return_, coef]) Simple detection of potential outliers based on interquartile range (IQR).
outliers_gesd(data[, outliers, report, alpha]) The generalized (Extreme Studentized Deviate) ESD test is used to detect one or more outliers in a univariate data set that follows an approximately normal distribution [R9f0a3d7f0aba-1].
outliers_grubbs(x[, hypo, alpha]) Grubbs’ Test for Outliers [R584a69a8f4e0-1].
outliers_tietjen(x, k[, hypo, alpha]) Tietjen-Moore test [Rdd1badcb01cb-1] to detect multiple outliers in a univariate data set that follows an approximately normal distribution.
